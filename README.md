# The DATATRAX

**He Speaks for the Data.**

A visual audiobook about data governance, told in the style of a beloved children's book — for adults who've sat in meetings where two dashboards showed two different headcounts.

## Play It

**[Read the audiobook here](https://barknard.github.io/datatrax-audiobook/)**

Works on desktop and mobile. Tap to play/pause, swipe to turn pages.

## Features

- 3 AI-generated character voices (via ElevenLabs)
- Word-by-word karaoke highlighting synced to audio
- 9 illustrated pages with SVG art
- Fully self-contained single HTML file — no server required
